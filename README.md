## This is Reversi game project for GUI lectures (in JavaFX).

![Image](reversi-game/resource/reversi.png)

### Project assumptions:

 * Easy to use UI
 * MVC design pattern

 ### Project features:
 
 * Timed Game (one click/double click -> on/off)
 * Load/Save state of game
 * Game summary
 * [TODO] AI Player module (based on MCTS)
