  `java -jar reversi-game-[version]-jar-with-dependencies.jar`

### Project assumptions:

 * Easy to use UI
 * MVC design pattern

 ### Project features:
 
 * Timed Game (one click/double click -> on/off)
 * Load/Save state of game
 * Game summary
